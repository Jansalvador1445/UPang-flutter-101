## Usage
### Download or Clone the project.
```
    git clone <Link here>
```

or Download the zip file : [Download](<Link here>)

### Run the Sample Project.
Mobile Devices
```
    cd UPang-flutter-101
    flutter pub get
    flutter run
```

Web Devices
```
    cd UPang-flutter-101
    flutter pub get
    flutter run -d chrome
```
